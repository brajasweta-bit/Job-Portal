# Job Portal

A full-stack Job Portal web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

## Features

### Candidate
- Register & Login
- Browse Jobs
- View Job Details
- Apply for Jobs
- Track Applications

### Employer
- Create Jobs
- Manage Job Listings
- Delete Jobs
- View Applicants
- Employer Dashboard

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Axios
